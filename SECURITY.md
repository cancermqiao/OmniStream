# Security Policy

## Supported Versions

This project is pre-1.0.0. Only the latest `main` branch is supported.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report privately to maintainers with:
- Affected component and version/commit
- Reproduction steps or proof of concept
- Potential impact
- Suggested remediation (optional)

The maintainers will acknowledge receipt within 3 business days and provide a remediation plan or timeline.
